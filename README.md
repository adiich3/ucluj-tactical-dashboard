# U Cluj Tactical Analysis Dashboard

Interactive football analytics dashboard built using Streamlit.

## Features

- Match tactical analysis
- Player performance tracking
- Team overview dashboards
- Opponent comparison
- Best Starting XI generator
- Formation visualization on pitch

## Installation

Clone the repository:

git clone https://github.com/USERNAME/ucluj-tactical-dashboard.git

Go into folder:

cd ucluj-tactical-dashboard

Install dependencies:

pip install -r requirements.txt

Run dashboard:

streamlit run dashboard_main.py
